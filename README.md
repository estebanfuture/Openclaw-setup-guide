# OpenClaw Setup Guide

Guía paso a paso para montar un entorno híbrido con:

- OpenClaw
- Ubuntu 24.04 en VirtualBox
- Gemini API
- Ollama en Windows host
- conexión entre la VM Ubuntu y Ollama local

## Objetivo

Documentar una instalación real, validada y reproducible, incluyendo:

- pasos correctos
- configuración funcional
- errores frecuentes
- soluciones verificadas

## Aviso importante

No subas nunca:

- API keys
- tokens
- archivos `.env`
- URLs del dashboard con credenciales
- capturas con datos sensibles
